Installation & erster Start

Drei Schritte, um Spendif.ai auf deinem Rechner zum Laufen zu bringen: herunterladen, installieren, erster Start. Alles offline, kein Konto nötig.

1Für dein System herunterladen

Die offiziellen Installer werden als Assets jedes GitHub Release veröffentlicht. Wähle das passende Format für dein Betriebssystem.

🍎

macOS

SpendifAi-*.dmg
DMG herunterladen
macOS 12 Monterey oder neuer. Apple Silicon + Intel.
🪟

Windows

SpendifAi-*.msix
MSIX herunterladen
Windows 10 1809+ / Windows 11. Sideload + vertrauenswürdiges Zertifikat in der Pre-Alpha erforderlich.
🐧

Debian / Ubuntu

spendifai_*_amd64.deb
.deb herunterladen
Ubuntu 22.04+, Debian 12+, Mint 21+.
🎩

Fedora / RHEL

spendifai-*.rpm
.rpm herunterladen
Fedora 40+, RHEL 9+, Rocky / Alma 9+.

Suchst du eine bestimmte Version? Siehe alle Releases · Quellcode: drake69/spendify.

2Installieren

macOS — DMG

1. Doppelklick auf die heruntergeladene .dmg-Datei.

2. Ziehe Spendif.ai.app in den Ordner Programme.

3. DMG auswerfen (Cmd+E) und die App über das Launchpad starten.

Falls "Spendif.ai kann nicht geöffnet werden, da der Entwickler nicht verifiziert werden kann" erscheint (Build noch nicht signiert): Rechtsklick auf die App → Öffnen → bestätigen.

(Drag & Drop vom DMG nach Programme)

Windows — MSIX

1. Doppelklick auf die heruntergeladene .msix-Datei.

2. Der in Windows integrierte App-Installer öffnet sich. Klicke Installieren.

3. Die App erscheint im Startmenü und wird von dort gestartet.

Bei "Unbekannter Herausgeber": PowerShell als Administrator öffnen und das Zertifikat einmalig importieren (siehe Anleitung).

📷 Screenshot

Linux — .deb / .rpm

Im Terminal:

sudo apt install ./spendifai_*.deb  (Debian/Ubuntu)

sudo dnf install ./spendifai-*.rpm  (Fedora/RHEL)

Das postinst-Skript installiert uv, erstellt das venv, lädt das Modell herunter und schreibt die .env. Vollständige Anleitung: install guide.

3Erster Start

Beim ersten Start richtet Spendif.ai die Umgebung ein — der längste Schritt, der aber nur einmal stattfindet.

⏱ Benötigte Zeit: 5–15 Minuten bei einer typischen Heim-Internetverbindung. Freier Speicherplatz: ~5 GB.
Lass das Fenster offen, bis der Fortschrittsbalken verschwindet.

3a. Splash + KI-Modell-Download

Per Doppelklick öffnet sich ein natives Fenster mit Splashscreen und Fortschrittsbalken. Spendif.ai erkennt verfügbares RAM/VRAM und lädt das passendste LLM herunter (Qwen 2.5, Gemma 3 — Größen von 2 bis 8 GB).

Das Modell liegt unter ~/.spendifai/models/ (macOS/Linux) oder %APPDATA%\Spendif.ai\models\ (Windows) und übersteht Neuinstallationen.

📷 Screenshot
assets/screenshots/splash-download-progress.png
(Splash + Fortschritt Modell-Download)

3b. Onboarding-Assistent — 4 Schritte

Sobald das Modell bereit ist, weicht der Splash dem Konfigurationsassistenten:

1. Sprache — Taxonomie, Datumsformat, Zahlentrennzeichen.
2. Inhaber — dein Name und die Varianten, die deine Bank verwendet.
3. Konten — Girokonto, Karten, Bargeld.
4. Bestätigen — alles wird erst hier geschrieben, keine teilweise Persistenz.

Keine Wahl ist endgültig: alles lässt sich später in den Einstellungen ändern.

📷 Screenshot
assets/screenshots/onboarding-step-1-language.png
(Assistent Schritt 1 — Sprachauswahl)

3c. App bereit

Nach der Bestätigung erstellt Spendif.ai die SQLite-Datenbank und führt dich zur Import-Seite. Von dort lädst du deine ersten Bank-CSV/XLSX-Dateien.

Ab dem nächsten Mal ist der Ablauf: Doppelklick → Splash (1-2 Sekunden) → App. Keine Downloads mehr, kein Assistent mehr.
