Install & first launch

Three steps to get Spendif.ai running on your computer: download, install, first launch. All offline, no account required.

✨What you get

  • Imports any bank's CSV/XLSX — auto-detects format, no manual mapping
  • AI categorisation, fully offline — local LLM included, no cloud needed
  • Local AI by default — data stays on your machine; remote LLM is opt-in and redacts IBAN / cards / owner names (amounts and dates still travel — see roadmap)
  • Five languages — Italian, English, French, German, Spanish
  • Card-account reconciliation (beta) — pairs credit-card settlements with the underlying expenses to avoid double-counting(edge cases still being refined)
  • Internal transfer detection (beta) — matches transfers across your own accounts so they don't inflate dashboards(edge cases still being refined)

2Install

macOS — DMG

1. Double-click the downloaded .dmg.

2. Drag Spendif.ai.app into the Applications folder.

3. Eject the DMG (Cmd+E) and launch the app from Launchpad.

If you see "Spendif.ai cannot be opened because the developer cannot be verified" (build not yet signed): right-click the app → Open → confirm.

Windows — MSIX

1. Double-click the downloaded .msix.

2. The built-in App Installer opens. Click Install.

3. The app appears in the Start Menu — launch from there.

If you see "Unknown publisher": open PowerShell as admin and import the certificate once (see the guide).

Linux — .deb / .rpm

From the terminal:

sudo apt install ./spendifai_*.deb  (Debian/Ubuntu)

sudo dnf install ./spendifai-*.rpm  (Fedora/RHEL)

The postinst installs uv, creates the venv, downloads the model and writes .env. Full guide: installation docs.

3First launch

On first launch Spendif.ai sets up its environment — it's the longest step, but happens only once.

⏱ Time required: depends on your connection speed (usually 5-15 minutes). Free disk space required: ~5 GB. Good news: you can start filling in the wizard right away, the model downloads in parallel.

3a. Splash + AI model download (in parallel)

Double-clicking opens a native window with a splash screen. Spendif.ai detects your RAM/VRAM and kicks off a background download of the LLM best suited for your hardware (Qwen 2.5, Gemma 3 — sizes from 1 to 8 GB).

The model lives at ~/.spendifai/models/ (macOS/Linux) or %USERPROFILE%\.spendifai\models\ (Windows) and survives reinstalls. The configuration wizard appears within seconds, without waiting for the download to finish.

3b. Onboarding wizard — 4 steps

While the model continues downloading in the background, the configuration wizard guides you through 4 steps:

1. Language — taxonomy, date format, number separators.
2. Holders — your name and the variants your bank uses.
3. Accounts — current account, cards, cash.
4. Summary + Start — everything is written only here, no partial persistence.

Every step shows a live download banner at the top (e.g. "📚 78% · ~3 min"). On the final summary, the Start button stays disabled until the model is 100% — so the Import page works immediately when you reach the app. Everything is changeable from Settings later.

📷 Screenshot

3c. App ready

On confirm the wizard creates the SQLite database and lands you on the Import page. From here you load your first bank CSV/XLSX files.

From then on the flow is: double-click → splash (1-2 seconds) → app. No more downloads, no more wizard.
