Installation et premier lancement

Trois étapes pour faire fonctionner Spendif.ai sur votre ordinateur : télécharger, installer, premier lancement. Tout en local, sans compte.

1Téléchargez pour votre système

Les installeurs officiels sont publiés en tant qu'assets de chaque GitHub Release. Choisissez le format adapté à votre OS.

🍎

macOS

SpendifAi-*.dmg
Télécharger DMG
macOS 12 Monterey ou supérieur. Apple Silicon + Intel.
🪟

Windows

SpendifAi-*.msix
Télécharger MSIX
Windows 10 1809+ / Windows 11. Sideload + certificat de confiance requis pendant la pré-alpha.
🐧

Debian / Ubuntu

spendifai_*_amd64.deb
Télécharger .deb
Ubuntu 22.04+, Debian 12+, Mint 21+.
🎩

Fedora / RHEL

spendifai-*.rpm
Télécharger .rpm
Fedora 40+, RHEL 9+, Rocky / Alma 9+.

Vous cherchez une version spécifique ? Voir toutes les releases · Code source : drake69/spendify.

2Installer

macOS — DMG

1. Double-cliquez sur le fichier .dmg téléchargé.

2. Glissez Spendif.ai.app dans le dossier Applications.

3. Éjectez le DMG (Cmd+E) et lancez l'app depuis Launchpad.

Si vous voyez "Impossible d'ouvrir Spendif.ai car le développeur ne peut pas être vérifié" (build pas encore signée) : clic droit sur l'app → Ouvrir → confirmer.

Windows — MSIX

1. Double-cliquez sur le fichier .msix téléchargé.

2. L'installeur d'applications intégré à Windows s'ouvre. Cliquez sur Installer.

3. L'app apparaît dans le menu Démarrer — lancez-la depuis là.

Si vous voyez "Éditeur inconnu" : ouvrez PowerShell en admin et importez le certificat une fois (voir le guide).

Linux — .deb / .rpm

Depuis le terminal :

sudo apt install ./spendifai_*.deb  (Debian/Ubuntu)

sudo dnf install ./spendifai-*.rpm  (Fedora/RHEL)

Le postinst installe uv, crée le venv, télécharge le modèle et configure .env. Guide complet : documentation d'installation.

3Premier lancement

Au premier lancement, Spendif.ai prépare son environnement — c'est l'étape la plus longue, mais elle n'arrive qu'une fois.

⏱ Temps requis : 5–15 minutes sur une connexion domestique typique. Espace libre requis : ~5 Go.
Gardez la fenêtre ouverte jusqu'à ce que la barre de progression disparaisse.

3a. Splash + téléchargement du modèle IA

Double-cliquer ouvre une fenêtre native avec un écran de démarrage et une barre de progression. Spendif.ai détecte la RAM/VRAM disponible et télécharge le LLM le plus adapté (Qwen 2.5, Gemma 3 — tailles de 2 à 8 Go).

Le modèle est stocké dans ~/.spendifai/models/ (macOS/Linux) ou %APPDATA%\Spendif.ai\models\ (Windows) et survit aux réinstallations.

3b. Assistant d'onboarding — 4 étapes

Une fois le modèle prêt, le splash laisse place à l'assistant de configuration :

1. Langue — taxonomie, format de date, séparateurs numériques.
2. Titulaires — votre nom et les variantes utilisées par la banque.
3. Comptes — compte courant, cartes, espèces.
4. Confirmer — tout est écrit ici uniquement, pas de persistance partielle.

Aucun choix n'est définitif : tout peut être modifié plus tard depuis Paramètres.

3c. App prête

À la confirmation, Spendif.ai crée la base SQLite et vous emmène sur la page Importer. Depuis là, vous chargez vos premiers CSV/XLSX bancaires.

Aux lancements suivants : double-clic → splash (1-2 secondes) → app. Plus de téléchargements, plus d'assistant.
