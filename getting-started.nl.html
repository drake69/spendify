Installatie & eerste start

Drie stappen om Spendif.ai op je computer te laten draaien: downloaden, installeren, eerste start. Alles offline, geen account nodig.

2Installeren

macOS — DMG

1. Dubbelklik op het gedownloade .dmg-bestand.

2. Sleep Spendif.ai.app naar de map Programma's.

3. Eject de DMG (Cmd+E) en start de app vanuit Launchpad.

Als je "Kan Spendif.ai niet openen omdat de ontwikkelaar niet kan worden geverifieerd" ziet (build nog niet ondertekend): rechtermuisklik op de app → Open → bevestigen.

Windows — MSIX

1. Dubbelklik op het gedownloade .msix-bestand.

2. De ingebouwde App Installer van Windows opent. Klik op Installeren.

3. De app verschijnt in het Startmenu — start hem daar vandaan.

Als je "Onbekende uitgever" ziet: open PowerShell als beheerder en importeer het certificaat eenmalig (zie de handleiding).

Linux — .deb / .rpm

Vanuit de terminal:

sudo apt install ./spendifai_*.deb  (Debian/Ubuntu)

sudo dnf install ./spendifai-*.rpm  (Fedora/RHEL)

De postinst installeert uv, maakt het venv aan, downloadt het model en schrijft .env. Volledige handleiding: install guide.

3Eerste start

Bij de eerste start zet Spendif.ai zijn omgeving op — dat is de langste stap, maar gebeurt slechts één keer.

⏱ Benodigde tijd: 5–15 minuten bij een typische thuisverbinding. Benodigde vrije ruimte: ~5 GB.
Houd het venster open tot de voortgangsbalk verdwijnt.

3a. Splash + AI-model download

Dubbelklikken opent een native venster met een splashscreen en voortgangsbalk. Spendif.ai detecteert beschikbaar RAM/VRAM en downloadt het meest geschikte LLM (Qwen 2.5, Gemma 3 — formaten van 2 tot 8 GB).

Het model staat in ~/.spendifai/models/ (macOS/Linux) of %APPDATA%\Spendif.ai\models\ (Windows) en blijft staan bij herinstallaties.

3b. Onboarding-wizard — 4 stappen

Zodra het model klaar is, maakt het splash plaats voor de configuratiewizard:

1. Taal — taxonomie, datumnotatie, numerieke scheidingstekens.
2. Houders — je naam en de varianten die je bank gebruikt.
3. Rekeningen — betaalrekening, kaarten, contant geld.
4. Bevestigen — alles wordt alleen hier geschreven, geen gedeeltelijke persistentie.

Geen enkele keuze is definitief: je kunt alles later wijzigen via Instellingen.

📷 Schermafbeelding
assets/screenshots/onboarding-step-1-language.png

3c. App klaar

Bij bevestiging maakt Spendif.ai de SQLite-database aan en brengt je naar de Import-pagina. Vanaf daar laad je je eerste bank-CSV/XLSX-bestanden.

Bij volgende keren is de flow: dubbelklik → splash (1-2 seconden) → app. Geen downloads meer, geen wizard meer.
