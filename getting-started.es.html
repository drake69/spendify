Instalación y primer inicio

Tres pasos para tener Spendif.ai funcionando en tu computadora: descargar, instalar, primer inicio. Todo offline, sin cuenta.

1Descarga para tu sistema

Los instaladores oficiales se publican como activos de cada GitHub Release. Elige el formato adecuado para tu SO.

🍎

¿Buscas una versión específica? Ver todas las releases · Código fuente: drake69/spendify.

2Instalar

macOS — DMG

1. Doble clic en el archivo .dmg descargado.

2. Arrastra Spendif.ai.app a la carpeta Aplicaciones.

3. Expulsa el DMG (Cmd+E) y lanza la app desde el Launchpad.

Si ves "No se puede abrir Spendif.ai porque no se puede verificar al desarrollador" (build aún no firmada): clic derecho en la app → Abrir → confirmar.

Windows — MSIX

1. Doble clic en el archivo .msix descargado.

2. Se abre el Instalador de Apps integrado en Windows. Pulsa Instalar.

3. La app aparece en el Menú Inicio y se lanza desde allí.

Si ves "Editor desconocido": abre PowerShell como admin e importa el certificado una vez (consulta la guía).

Linux — .deb / .rpm

Desde la terminal:

sudo apt install ./spendifai_*.deb  (Debian/Ubuntu)

sudo dnf install ./spendifai-*.rpm  (Fedora/RHEL)

El postinst instala uv, crea el venv, descarga el modelo y configura .env. Guía completa: guía de instalación.

3Primer inicio

En el primer inicio Spendif.ai prepara el entorno — es el paso más largo, pero ocurre solo una vez.

⏱ Tiempo requerido: 5–15 minutos en una conexión doméstica típica. Espacio libre requerido: ~5 GB.
Mantén la ventana abierta hasta que desaparezca la barra de progreso.

3a. Splash + descarga del modelo de IA

Al hacer doble clic se abre una ventana nativa con una pantalla splash y barra de progreso. Spendif.ai detecta la RAM/VRAM disponible y descarga el LLM más adecuado (Qwen 2.5, Gemma 3 — tamaños de 2 a 8 GB).

El modelo se guarda en ~/.spendifai/models/ (macOS/Linux) o %APPDATA%\Spendif.ai\models\ (Windows) y sobrevive a reinstalaciones.

📷 Captura
assets/screenshots/splash-download-progress.png
(splash + progreso descarga modelo)

3b. Asistente de onboarding — 4 pasos

Cuando el modelo está listo, el splash da paso al asistente de configuración:

1. Idioma — taxonomía, formato de fecha, separadores numéricos.
2. Titulares — tu nombre y las variantes que usa tu banco.
3. Cuentas — cuenta corriente, tarjetas, efectivo.
4. Confirmar — todo se escribe solo aquí, sin persistencia parcial.

Ninguna elección es definitiva: puedes cambiar todo después desde Ajustes.

📷 Captura
assets/screenshots/onboarding-step-1-language.png
(asistente paso 1 — selección idioma)

3c. App lista

Al confirmar el asistente, Spendif.ai crea la base de datos SQLite y te lleva a la página Importar. Desde ahí cargas tus primeros CSV/XLSX bancarios.

A partir del siguiente inicio el flujo es: doble clic → splash (1-2 segundos) → app. Sin más descargas, sin más asistente.
