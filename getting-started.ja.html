インストールと初回起動

Spendif.ai をお使いのコンピューターで動かすための 3 ステップ: ダウンロード、インストール、初回起動。すべてオフライン、アカウント不要です。

1お使いの OS 用にダウンロード

公式インストーラーは各 GitHub Release のアセットとして公開されています。OS に合ったフォーマットを選んでください。

🍎

macOS

SpendifAi-*.dmg
DMG をダウンロード
macOS 12 Monterey 以降。Apple Silicon + Intel 対応。
🪟

Windows

SpendifAi-*.msix
MSIX をダウンロード
Windows 10 1809+ / Windows 11。プレアルファ期間中はサイドロード + 信頼された証明書が必要です。
🐧

Debian / Ubuntu

spendifai_*_amd64.deb
.deb をダウンロード
Ubuntu 22.04+, Debian 12+, Mint 21+。
🎩

Fedora / RHEL

spendifai-*.rpm
.rpm をダウンロード
Fedora 40+, RHEL 9+, Rocky / Alma 9+。

特定のバージョンをお探しですか? すべてのリリースを参照 · ソースコード: drake69/spendify。

2インストール

macOS — DMG

1. ダウンロードした .dmg ファイルをダブルクリックします。

2. Spendif.ai.app を アプリケーション フォルダにドラッグします。

3. DMG を取り出し (Cmd+E)、Launchpad からアプリを起動します。

「開発元を確認できないため、Spendif.ai を開けません」と表示された場合 (まだ署名されていないビルド): アプリを右クリック → 開く → 確認。

(DMG からアプリケーションへドラッグ & ドロップ)

Windows — MSIX

1. ダウンロードした .msix ファイルをダブルクリックします。

2. Windows 内蔵のアプリ インストーラーが開きます。インストール をクリックします。

3. アプリがスタートメニューに表示されます — そこから起動します。

「不明な発行元」と表示された場合: 管理者として PowerShell を開き、証明書を一度インポートしてください (ガイド参照)。

📷 スクリーンショット

Linux — .deb / .rpm

ターミナルから:

sudo apt install ./spendifai_*.deb  (Debian/Ubuntu)

sudo dnf install ./spendifai-*.rpm  (Fedora/RHEL)

postinst が uv をインストールし、venv を作成し、モデルをダウンロードして .env を構成します。完全なガイド: インストールドキュメント。

3初回起動

初回起動時、Spendif.ai は環境をセットアップします — 一番時間がかかるステップですが、一度だけです。

⏱ 所要時間: 一般的な家庭用回線で 5〜15 分。必要な空き容量: 約 5 GB。
プログレスバーが消えるまでウィンドウを開いたままにしてください。

3a. スプラッシュ + AI モデルのダウンロード

ダブルクリックで、スプラッシュ画面とプログレスバーを備えたネイティブウィンドウが開きます。Spendif.ai は利用可能な RAM/VRAM を検出し、最も適した LLM をダウンロードします (Qwen 2.5、Gemma 3 — 2〜8 GB)。

モデルは ~/.spendifai/models/ (macOS/Linux) または %APPDATA%\Spendif.ai\models\ (Windows) に保存され、再インストールしても残ります。

📷 スクリーンショット
assets/screenshots/splash-download-progress.png

3b. オンボーディング ウィザード — 4 ステップ

モデルの準備ができると、スプラッシュは 設定ウィザード に切り替わります:

1. 言語 — 分類体系、日付形式、数値区切り。
2. 名義人 — あなたの名前と銀行が使うバリエーション。
3. 口座 — 当座預金、カード、現金。
4. 確認 — すべてここでのみ書き込まれ、部分的な永続化はありません。

どの選択も最終的ではありません: 後から 設定 で変更できます。

📷 スクリーンショット
assets/screenshots/onboarding-step-1-language.png
(ウィザード ステップ 1 — 言語選択)

3c. アプリ準備完了

ウィザードの確認時に、Spendif.ai は SQLite データベースを作成し、インポート ページに移動します。そこから最初の銀行 CSV/XLSX ファイルを読み込みます。

次回以降の起動フローは: ダブルクリック → スプラッシュ (1〜2 秒) → アプリ。ダウンロードもウィザードもありません。
