Instalacja i pierwsze uruchomienie

Trzy kroki, aby uruchomić Spendif.ai na komputerze: pobierz, zainstaluj, pierwsze uruchomienie. Wszystko offline, bez konta.

1Pobierz dla swojego systemu

Oficjalne instalatory są publikowane jako zasoby każdego GitHub Release. Wybierz format odpowiedni dla Twojego systemu.

🍎

macOS

SpendifAi-*.dmg
Pobierz DMG
macOS 12 Monterey lub nowszy. Apple Silicon + Intel.
🪟

Windows

SpendifAi-*.msix
Pobierz MSIX
Windows 10 1809+ / Windows 11. W fazie pre-alfa wymagany sideload + zaufany certyfikat.
🐧

Debian / Ubuntu

spendifai_*_amd64.deb
Pobierz .deb
Ubuntu 22.04+, Debian 12+, Mint 21+.
🎩

Fedora / RHEL

spendifai-*.rpm
Pobierz .rpm
Fedora 40+, RHEL 9+, Rocky / Alma 9+.

Szukasz konkretnej wersji? Zobacz wszystkie wydania · Kod źródłowy: drake69/spendify.

2Instaluj

macOS — DMG

1. Kliknij dwukrotnie pobrany plik .dmg.

2. Przeciągnij Spendif.ai.app do folderu Programy.

3. Wysuń DMG (Cmd+E) i uruchom aplikację z Launchpada.

Jeśli widzisz "Nie można otworzyć Spendif.ai, ponieważ nie można zweryfikować dewelopera" (kompilacja jeszcze niepodpisana): kliknij prawym przyciskiem na aplikację → Otwórz → potwierdź.

Windows — MSIX

1. Kliknij dwukrotnie pobrany plik .msix.

2. Otworzy się wbudowany Instalator Aplikacji Windows. Kliknij Zainstaluj.

3. Aplikacja pojawi się w Menu Start — uruchom ją stamtąd.

Jeśli widzisz "Nieznany wydawca": otwórz PowerShell jako administrator i zaimportuj certyfikat jednorazowo (zobacz przewodnik).

Linux — .deb / .rpm

Z terminala:

sudo apt install ./spendifai_*.deb  (Debian/Ubuntu)

sudo dnf install ./spendifai-*.rpm  (Fedora/RHEL)

postinst instaluje uv, tworzy venv, pobiera model i konfiguruje .env. Pełny przewodnik: dokumentacja instalacji.

3Pierwsze uruchomienie

Przy pierwszym uruchomieniu Spendif.ai konfiguruje środowisko — to najdłuższy krok, ale wykonywany tylko raz.

⏱ Wymagany czas: 5–15 minut na typowym domowym łączu. Wymagane wolne miejsce: ~5 GB.
Trzymaj okno otwarte, dopóki pasek postępu nie zniknie.

3a. Splash + pobieranie modelu AI

Podwójne kliknięcie otwiera natywne okno z ekranem powitalnym i paskiem postępu. Spendif.ai wykrywa dostępny RAM/VRAM i pobiera najbardziej odpowiedni LLM (Qwen 2.5, Gemma 3 — rozmiary od 2 do 8 GB).

Model trafia do ~/.spendifai/models/ (macOS/Linux) lub %APPDATA%\Spendif.ai\models\ (Windows) i przetrwa reinstalacje.

📷 Zrzut ekranu
assets/screenshots/splash-download-progress.png

3b. Kreator wprowadzający — 4 kroki

Gdy model jest gotowy, splash ustępuje miejsca kreatorowi konfiguracji:

1. Język — taksonomia, format daty, separatory liczbowe.
2. Właściciele — Twoje imię i warianty używane przez bank.
3. Konta — konto bieżące, karty, gotówka.
4. Potwierdź — wszystko jest zapisywane tylko tutaj, brak częściowej trwałości.

Żaden wybór nie jest ostateczny: możesz zmienić wszystko później w Ustawieniach.

3c. Aplikacja gotowa

Po potwierdzeniu Spendif.ai tworzy bazę SQLite i przenosi Cię do strony Import. Stamtąd ładujesz pierwsze pliki CSV/XLSX z banku.

Przy kolejnych uruchomieniach przebieg to: podwójne kliknięcie → splash (1-2 sekundy) → aplikacja. Bez pobierania, bez kreatora.
